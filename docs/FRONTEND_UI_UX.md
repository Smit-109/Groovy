# UI/UX Specifications

**Design Language**: Dark modern (as in project HTML) — deep purples, teals, golds.

**Key Screens**:

1. **Landing / Login**
2. **Student Dashboard** — Enrolled courses, progress rings, recommended next lesson
3. **Course Viewer** — Sidebar modules, main content area, "Ask AI Tutor" floating button
4. **AI Chat Modal** — Clean chat with sources
5. **Instructor Dashboard** — Student list, heatmaps of weak topics, completion rates
6. **Quiz Interface** — Beautiful question cards with instant feedback

**Components**: Use shadcn/ui (Button, Card, Progress, Avatar, Tabs, etc.)
